# Enchanté *(miło mi)*

## Exercice 1 : traduisez

1. Nazywam się Marek Nowak.
   Je m'appelle Marek Nowak.
2. Miło mi.
   Enchanté. 
3. Dzień dobry.
   Bonjour.
4. Ja też.
   Moi aussi.

## Exercice 2 : complétez

1. Bonjour.

   Dzień **dobry**.

2. Enchantée.
   **Miło** mi.

3. Je m'appelle Kowalska.
   Nazywam **się** Kowalska.

4. Moi aussi.
   **Ja** też.
